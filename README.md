# ACM Gene Plot Viewer

This repository provides an interactive browser for viewing all PDF plots generated for the ACM project.  
The website allows users to quickly search for a gene of interest and view corresponding plots without loading all files at once.

## Features

- Fast loading: PDF files are not loaded until requested.
- Searchable: Type any gene name to instantly retrieve its plots.
- Direct download: Each plot can be downloaded as an individual PDF.
- Lightweight and static: Fully supported by GitHub Pages.

---

## How to Use

1. Open the viewer:  
   **https://jiehuichen.github.io/ACM_plots/**

2. Type a gene name (e.g., `Acvr1`, `Nlrp3`, `Cdh11`) in the search box.

3. The viewer will display:
   - A large PDF preview of each matching plot  
   - A direct download link for each file

4. No installation is required.
